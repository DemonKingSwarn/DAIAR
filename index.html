Demon's Arch Installer and Auto Ricer (DAIAR)

Installation:

            
pacman -Sy git archlinux-keyring
git clone --depth 1 https://github.com/demonkingswarn/archinstall
cd archinstall && ./install.sh

That's it.

What is DAIAR?

DAIAR is a script that autoinstalls and autoconfigures a fully-functioning and minimal Arch Linux environment.

DAIAR can be run on a fresh install of Arch Linux, providing you with a fully configured diving-board for work or more customizations.

Customization

DAIAR uses the programs here in pkgs.txt and installs my dotfiles repo here, but you can easily modify it by editing the script.

Formatting a Disk Guide

Follow these steps to format a disk:

            
# List available block devices
lsblk

# Assume the disk is named 'sda'
fdisk /dev/sda

# Press 'm' for help
m

# Create a new GPT partition table
g

# Create the first partition (550MB)
n



+550M

# Create the second partition (2GB)
n




+2G

# Set the first partition type to EFI System
t
1
1

# Set the second partition type to Linux Swap
t
2
19

# Write changes and exit
w
